TypeScript

TypeScript yra JavaScript superset'as

Type script papildo JavaScript kalbą įvairiais tipais. Leidžia programuotojui nusttinėti kintamųjų, parametrų, grąžinimo... tipus. (pvz.: Pasakyti, kad kintamasis privalo būti string tipo)

TypeScript labai greit išpopuliarėjo apie 2018 kaip JS kalbos superset'as, dėl jo naudos dirbant komandose ir/ar prie didelių tobulinimų projektų.

TypeScript tipų nurodymas naudingas, nes ji gaudo įvairiais llogine klaidas jau kodo rašymo metu, vitoj to, kad klaidos būtų atvaizduojamos kodo vykdymo metu.

Puslapis nesupranta TS tipo failų, dėl to prie HTML'o visiem reikia prikabinti JS failus. Tam, kad galėtume naudoti TS kodą savo puslapiuose, jį reikia kopiliuoti/konvertuoti į JS kodą. Tam, kad taip padarytume, reikia naudoti tsc komandą, kurią galime naudoti tik tuomet, jeigu mūsų projekte/įrenginyje yra įrašytas typescript'as ir jo config'as.

Set-up

TypeScript naudojimo įvadas

  1. įsirašyti node.js
  2. įsirašyti TS globaliai (npm install -g typescript)
  3. susikurti TS failą su projekte ir jame rašyti kažkokį TS kodą
  4. Susikurti TS config failą: tsc --init
  5. tsconfig.json faile pasikeisti norimus nustatymus
  6. sukompiliuoti TS failą į JS failą: tsc --watch main.ts

Es-Lint

  1. įsirašyti Es-Lint (globaliai arba lokaliai)
  2. pasikurti projekto package.json failą: npm init
  3. susikurti Es-Lint config failą: eslint --init
  4. pasikeisti norimus Es-Lint config nustatymus
  5. įsirašyti ir įgalinti ESlint papildinį (exyension) VS Code
