Union

OR - |

Nurodo, kad galime priskirti vieną arba kitą arba trečią... tipą, kitaip pasakius vieną iš išvardintų tipų
(pvz.: type StringOrNumber = string | number);

AND - &

Nurodo, kad reikia paskirti IR vieną IR kitą IR trečią... tipus, kitaip pasakius visus išvadintus
(pvz.: type Asmuo = {vardas: string, pavarde: string}; type AsmuoBonus = {amzius: number, ugis:number}; type PilnasAsmuo = Asmuo & AsmuoBonus;)

(Type1 | type2)[]

Jeigu norime mi6raus masyvo, reikia jo galimus tipus apra6yti prie6 masyvo sklaistus, paprastuose skliausteliuose vardinant tipus su OR simboliu
(pvz.: type StringIrNumberArray = (string | number)[];)

Tuple

Tuple yra griežtos struktūros masyvas. Yra nurodomas masyvo ilgis (pagal tai kiek tipų jo viduje išvardinta) ir kiekvienas jo elementas privalo turėti nuosavą tipą (tipai gali būti skirtingi)
(pvz.: type Coordinates = [number, number];)

Literal Types

Naudojami norint sukonkretinti funkcijos parametro galimus variantus arba sukonkretinti objekto savybės gaimus variantus. Tokiu būdu duodame pasirinkti tik vieną iš aprašytų variantų, vietoj to kad rašyti betką iš string ar nuber ar boolean reikšmių.
(pvz.: type TShirt_size = 'small' | 'medium' | 'large';)

Enum

enum - enumerator yra skirtas sukurti pasirinkimo variantų sąrašą. Sąrašas turi pavadinimus ir už kiekvieno pavainimo slypi skiačius (kuris yra generuojamas eilės tvarka automatiškai) arba kažkokia reikšmė (kurią pats nusirodai kurdamas enum'ą).
Jo nauda yra tokia kaip ir literal types, kad nebūtų netyčia panaudojamos/priskiriamos neegzistuojančios reikšmės.

Class

Klasės TS's yra aprašomas šiek tiek kitaip negu JS'e.
TS'e prieš konstruktorių, mes TURIME susikurti savybes (properties) su tipais ir taip pat galime priskirti joms prieinamumus (public, private arba readonly).

Prieinamumai (prefix modifiers)

  • public (default) - nieko nekeičia
  • private - pasiekiamas tik Klasės viduje. Norint pasiekti už klasės NEBAIGTA RASYT
