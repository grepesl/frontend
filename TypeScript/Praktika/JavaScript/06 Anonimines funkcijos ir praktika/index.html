Anoniminės funkcijos

Tai funkcijia, kuri neturi savo pavadinimo. Dažnu atveju tokia funkcija rašoma toje vietoje, kur jinai iš karto yra iškviečiama.

Yra toks terminas kaip "callback" kuris dažnu atveju būna anoniminė funkcija, kuri yra rašoma kaip argumentas kitos funkcijos iškvietime.

Anoniminių funkcijų variantai

  • įprasta anoniminė: function(parametrai){ return }
  • įprasta anoniminė prilyginta: const pav = function(par){ return }
  • arrow/lambda: (par) => { veiksmai su return }
  • arrow.lambda: su vienu veiksmu: (par) => par*3
  • arrow/lambda prilyginta: const pav = (par) => { veiksmai su return }
  • anoniminė save išsikviečianti: (function(par){ veiksmai })()
