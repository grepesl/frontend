Funkcija

Funkcija yra užvadintas kodas, kuris nėra vykdomas kol į jį bėra kreipiamasi. Galima kreiptis daugiau nei veiną kartą.

Funkcija GALI priimti duomenis. Tuos duomenis naudoja savo viduje atliekant veiksmus.

Funkcija gali atlikti kazkoki veiksma ir/arba grąžinti kažkokius duomenis.

Parametras - funkcijos kūrimo metu aprašytas kintamasisfunkcijos pradiniose sklaisutuose, kurio reikšmė priklauso nuo to, kas bus paduota lreipimosi į funkciją metu kaip argumentas.
Argumentas - reikšmė kuri yra paduodama į funkciją kreipimosi metu.

Syntax

function susigalvotasPavadinimas(){... veiksmas ...}
function susigalvotasPavadinimas2(parametras1, par2, par3...){... veiksmai ...}

Call

susigalvotasPavadinimas(); susigalvotasPavadinimas2(argumentas, arg2, arg3, ...);

DOM

DOM - document Pbject model. Kaip HTMLą mato JS'as.

Visas HTML dokuimentas yra tarsi didelis objektas, kurio viduje yra 5vair8s raktioniai žodžiai ir juos gali būti masyvai kitų HTML elemntų.
pvz.: const html={children: [head={}, body={children: [header={}, main={...}, footer={}]}]}

Event

Kreipimasis į dokumento elemntą ir uždėjimas jam įvykio klausimąsį. Tame elemente įvykus tam įvikiui, bus atliekami kažkokie veiksmai.

  1. const elementas = document.querySelector('CSS_SELECTOR');
  2. elementas.addEventListener('įvykis', functionCall);
  3. function functionName(){... veiksmai ...};
