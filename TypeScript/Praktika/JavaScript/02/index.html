Priskyrimo (assignment operators)

Kintamųjų reikšmės kodo eigoje keičiasi tam, kad jas pakeistume reikia kintamajį prilyginti naujai reikšmei. Tai galima padaryti įvairiais būdais:

  • kintamasis++; - padidina kitamjojo reikšmę vienetu
  • kintamasis--; sumažina kitamjojo reikšmę vienetu
  • ++kintamasis; - padidina kitamjojo reikšmę vienetu
  • --kintamasis; sumažina kitamjojo reikšmę vienetu
  • kintamasis = nauja_reiksme; - prilygina kintamąjį naujai reikšmei
  • kintamasis = kintamasis + papildymas; - papildo kintamąjį papildoma reikšme
  • kintamasis += papildymas; (įdentiškas užrašymas viršutiniam ^)
  • kintamasis += papildymas; - sumažina kintamąjį papildoma reikšme
  • kintamasis *= daugiklis;
  • kintamasis /= daliklis;
  • kintamasis %= daliklis (kas lieka po dalybos);
  • kintamasis **= laipsnis
  • kintamasis **=

Palyginimai

Palygiminai yra naudojami tam, kad galėtume patikrinti ar reikšmės yra lygios/nelygios ar didesnės/mažesnės už vieną kitą.

  • x > y - ar x yra daugaiu už y
  • x < y - ar x yra mažiau už y
  • x >= y - ar x yra daugiau arba lygus y
  • x <= y - ar x yra mažiau arba lygus y
  • x == y - ar x lygus y (tikrina tik reikšmes)
  • x === y - ar x lygus y (tikrina reikšmes ir tipus)
  • x != y - ar x nelygus (reikšmės)
  • x !== y - ar x nelygus y (reikšmė ir tipai)

Loginiai (loginiai operatoriai)

  • ! - NOT - apverčia reikšmę
  • || - OR - reikalauja, kad abejose pusėse būtų melas, kad grąžintų melą, kitu atveju grąžins tiesą.
  • && - AND - reikalauja, kad abėjose pusėse būtų tesa, kad grąžintų tiesą, kitu atveju grąžins melą

Tipo (type operators)

  • typeof(kazkas) - kokio duomens tipo yra kažkas - grąžinamas atsakymas teksto pavidalu
  • object instanceof Class - ar pbjektas buvo sukurtas naudojant tą Klasę(Class)

Masyvai (array) ir Objektai

Jau mokinomės string, number, boolean tipo kintamuosius. Šiandien susipažinsime su Array (masyvas) ir Object(objektas), kurie yra konteineriniai kintamieji, kuriuose galima talpinti daugeli kitų reikšmių (net ir pačius konteinerius)

Array

Array - masyvas skirtas grupuoti kintamuosius.
Sintaksė: ['zodis',654,false,false,3,21,'strings',...]
Elementus masyvo viduje pasiekiame (kreipiames) nurodydami ju eiliškumo numerį. Masyvo eiliškumas skaičiuojamas nuo 0.
Į masyvo elementus kreipiamės nurodydami elemento eiliškumo skaičiu laužtiniuose skaliusteliuose už masyvo pavadinimo. (masyvoVardas[numeris])
Eiliškumas yra svarbus masyve.
Naudojami norint _IŠVARDINTI_ kažkokius susijusius dalykus.

Object

Object - objektas naudojamas grupuoti kintamuosius.
Sintaksė: {raktinisZodis:"reiksme",kitasRaktinisZodis:"kitaReiksme"...}
Elementus objekto viduje pasiekiame (kreipiames Į JUOS) nurodydami raktinį žodį po objekto pavadinimo padėjus tašką. (objektoPavadinimas.raktinisZodis)
Eiliškumas NĖRA svarbus.
Naudojamas norint kažkokį vieną dalyką _APRAŠYTI_.

Sąlygos

Principas - jeigu sąlygos klausimas grąžina tiesą - sąlygos vykdomojo bloko dalies kodas yra vykdomas.

Variacijos

  • if - pagrindinė sąlyga
    1. else if - papildoma sąlyga
    2. else if - papildoma sąlyga
    3. else if - papildoma sąlyga
    4. ...
    5. else - likę atvejai.
  • switch
  • ternary
  • loginio && sąlygio

If'ai

If

if(sąlyga){... veiksmai vykdomi tiesos atveju ...}

If else_if

if(sąlyga1){ ... veiksmai vykdomi sąlyga1 tiesos atveju ... } else if(sąlygą2){ ... vaiksmai vykdomi sąlygai yra melas ir sąlyga2 yra tiesa }

if else

if(sąlyga){ ... veiksmai vykdomi sąlyga tiesos atveju... } else{ ... veiksmai vykdomi sąlyga melo atveju... }

If else_if else

if(sąlyga1){ ... veiksmai vykdomi sąlyga1 tiesos atveju ... } else if(sąlygą2){ ... vaiksmai vykdomi sąlygai yra melas ir sąlyga2 yra tiesa } else { ... veiksmai vykdomi sąlyga1 ir sąlyga2 melo atveju... }
