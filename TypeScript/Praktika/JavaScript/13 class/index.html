Class (JS)

Klasė yra objektų kūrimo šablonas/brėžinys, kurį galime naudoti daug kartų norint sukurti daug objektų.
Klasės viduje esančiame konstruktoriuje (constructor) reikia išvardinti savybes (properties), kurias nori, kad objektai, sukurti iš tos klasės, turėtų.
Kreipdamiesi į klasę ir prašydami, kad mum sukurtų naują (new) objektą, galime paduoti argumentus, pagal kuriuos tas naujas objektas bus sukurtas. Tokiu būdu objektai nebus identiški.

Klasė privalo būti aprašyta prieš kreipianti į ją

Norėdami sukurti naują objektą iš klasės, kreipimosi metu TURIME rašyti žodelį "new".

OOP (Object Orientated Programming) principai

  • Encapulation - Inkapsuliacija
  • Abstraction - Abstrakcija
  • Inheritance - Paveldėjimas
  • Polymorphism - Polimorfizmas
