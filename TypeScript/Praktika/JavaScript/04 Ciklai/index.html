Ciklai

Ciklai yra naudojami norint atlikti kažkokius (vienodus/panašius) veiksmus daugelį kartų.

Ciklų variacijos

  • for(skaitliukoSukurimas,sąlygaSuSkaitliuku,skaitliukoKitimas){... veiksmai ...} - naudojams kai galima apskaičiuoti ciklų kiekį
  • while(sąlyga){... veiksmai ...} - naudojamas kai negalima apskaičiuoti ciklų kiekio
  • do{... veiksmai ...}while(sąlyga) - tas pats kas while tik veiksmai visada įvyks bent vieną kartą
  • for(let el of array){... veiksmai ...} - suka ciklą per array prilygindamas el vienai masyvo reikšmei vienos interacijos metu
  • for(let el in object){... veiksmai ...} - suka ciklą per object prilygindamas el vieną tos iteracijos raktinį žodį
  • masyvo iteraciniai metodai

Ciklas / Cycle / Loop -

Ciklai yra vykdomi tol kol yra tenkinama salyga

  • Gali vykti numatytą kiekį kartų (dažniausiai for)
  • Gali vykti nenumatytą kiekį kartų (dažniausiai while)
  • Gali vykti begalybę kartų - BLOGAI. Kai nurodoma neteisinga sąlyga / nekinta sąlyga.

Dažniausi ciklų panaudojami atvejai

  • Sukti ciklą per masyvą ir imant po vieną reikšmę (dažniasuiai objektą) atlikti kažkokius veiksmus
  • Sukti ciklą skaičių diapazone ir ieškoti atitinkamų sakičių
  • Sukti ciklą tol kol kažkas įvyksta
  • ...

Funkcija

Funkcija yra užvadintas kodas, kuris nėra vykdomas kol į jį bėra kreipiamasi. Galima kreiptis daugiau nei veiną kartą.

Funkcija GALI priimti duomenis. Tuos duomenis naudoja savo viduje atliekant veiksmus.

Funkcija gali atlikti kazkoki veiksma ir/arba grąžinti kažkokius duomenis.

Syntax

function susigalvotasPavadinimas(){... veiksmas ...}

Call

susigalvotasPavadinimas();
