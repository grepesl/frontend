Komentuoti galima paspaudus CTRL + /

kodą reikėtų komentuoti. kad kodas būtų suprantamas jums po kurio laiko arba kitam programuotojui - reikia pakomentuoti funkciją/salygą/ciklą/klasę... kad nereikėtų pačiam nagrinėtis/aiškintis ką jinai daro, o paskaičius trumpą komentarą - būtų aišku.

JS rašymo būdai

  • Internal - JS rašymas HTML failo viduje, script tago viduje
  • Externnal - JS rašymas atskirame JS faile, kuris yra prijungtads prie htmlo

Pagrinde naudojamas external rašymo būdas, jo prijungimas rašomas head tago viduje. Kadangi HTML failas yra skaitomas paeiliui nuo viršaus į apačią gaunasi, kad JS perskaitomas prieš HTMLą, O būna, kad JS kreipiasi į HTML bodį esančią info kuri dar nėra perskaityta ir tokiu atveju gauname kalidą dėl eiliškumo. Tam kad išvengti topkių atvejų JS prijungime rašomas "defer" atributas, kuris nukelia to failo skaitymą į pabaigą.

Informacijos atvaizdavymas naršyklėje

  • console.log("pranešimas"); - išvedimas į knsole
  • console.dir(HTML_Element(nodeElement)); - išsamus išvedimas į knsole
  • console.group("pavadinimas"); - sukuria atvirą konsolės grupę nurodytu pavadinimu
  • console.groupCollapsed("pavadinimas"); - sukuria suskleistą konsolės grupę nurodytu pavadinimu
  • console.groupEnd(); - nurodo konsolės grupės pabaigą
  • window.alert("pranešimas"); - perspėjimo lentelės atvaizdavimas ekrane
  • window.prompt("užklausa"); - puslapyje atvaizduojams pranešimas su vieta atsakymui, kuris yra grąžinamas į kodą teksto pavidalu
  • Ateityje: innerHTML / textContent / innerText / append / prepend / appendChild...

Kintamieji

Susigalvotas vardas, kurio viduje yra talpinama kokia nors informacija. Kreipiantis į tą vardą (kintajį) yra išsaugoma jo viduje esanti informacija, kurią po to kažkur galima panaudoti.

Kintamajo kūrimas

  1. kintamojo startas
    • var - senasis būdas (hoist'ingo pliusas)
    • let - naujasis būdas gali kisti
    • const - naujasis būdas negali kisti
    • - (nieko nerašant) !!nenaudoti!!
  2. kintamojoVardas - susigalvoji logišką, ne per ilgą kintamojo vardą be tarpų ar lietuviškų simbolių
  3. = - lygybės/priskyrimo simbolius
  4. reikšmė
    • Number - 6|-5|5.1|0|654321|21-4|...
    • String - "dvigubose"|'viengubose'|"du žodžiai"|'backtick'|...
    • Boolean - true|false
    • ...
  5. ; - loginės eilutės pabaiga

Arimetrika

Atliekami kažkokie veiksmai tarp kinatmųjų.
Galima atlikti veiksmus tarp skirtingo tipo kintamųjų.

  • x + y - sudėtis
  • x - y - atimtis
  • x * y - daugyba
  • x / y - dalyba
  • x % y - liekana
  • x ** y - laipsnio kėlimas
  • x ** (1/y) - šaknies traukimas

Visi arimetriniai veiksmai APART sudėties gali būti atliekami tik su skaičiais (su keliomis išimtimis)

  • String +string = string
  • string + number = string
  • boolean + string = string
  • boolean + number = number
  • boolean + boolean = number
