CRUD

Create Read Update Delete egzistuoja beveik visuose puslapiuose vienu ar kitu formatu. Labai daug kur vartotojai gali skaityti sugeneruotą informaciją, susikurti paskyras, post'us, siūsti žinutes, jas trinti ar redaguoti.
Nebūtinai būna pilnas CRUD'as, kartais gali būti tiktais kelios jo dalys, bet jos visviena būna beveik visur.

  • Create - sukurti naują: vartotoją/žinutę/post'ą...
  • Read - atsisiūsti ir atvaizduoti kažkokią informaciją
  • Update - redaguoti kažkokią atvaizduotą informaciją vienu ar kitu būdų (galimai atidarant redagavimo lauką kaip modalą)
  • Delete - trinti kažkokį informacijos vienetą (dažnu atveju su patvirtinimu)
