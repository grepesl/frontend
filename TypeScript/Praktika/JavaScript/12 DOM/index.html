DOM - Document Object Model

Node

Node'as HTML elementas JS'o aplinkoje. Jis yra objektas su labai daug savybių ir metodų

Galima HTNL'ą pildyti innerHTML'u - rašant HTML kodą JS kaip stringą ir tuo stringu papildyti HTML'ą (įskiepyti/injectinti HTML kodą). Tuomet tas string'as HTML'e bus skaitomas kaip HTML kodas ir atvaizduojamas kaip atitinkami HTML elementai.
Šitas būdas turi daug minusų, pradedant nuo saugumo, pabaigiant implementacija.

Geresnis būdas yra pildyti HTML'ą su JS'u - kurti node'us (HTML elementus) juos prilyginti kintamiesiems ir tuomet tuos node'us append'inti (patalpinti) į kitus HTML elementus.
Node'ai yra kuriami , modifikuojami ir pridedami į HTML'ą naudojant Node'ų metodus.

Node Metodai

  • document.createElement('tagName')
  • document.createElement('tekstas koks tik nori...')

  • kam.prepend(ką0, ką1, ką2...) - į kokį node'ą (kam) pradžioje įdėsime kokį'ius node'ą/node'us/string'ą/string'us... (ką)
