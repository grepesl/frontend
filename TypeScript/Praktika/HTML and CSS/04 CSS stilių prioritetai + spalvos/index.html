Spalvos gali būti aprašytos įvairiais būdais. Nuo žodžių pavd, kurie aiškūs eiliniem vartotojui, iki kažkokių neaiškių kodų, kuriuos supranta ne kiekvienas.

Spalvos

    Spalvos pavadinimas - vos 140 užvadintų spalvų.
  • RGB - trijų spalvų kontrastas (0-250).
  • Hex - trijų spalvų kontrastas (00-FF).
  • HSL - atspalvis, sodrumas, šviesumas.

Spalvos vardas

Spalva suteikiama nurodant jos vardą.

RGB

Susideda iš red green blue spalvų. Ir to pasakoje išgauname ~16,7m spalvų
RGB gali turėti permatomumą (opacity) (alpha) tuomet kodas vadinamas RGBA ir turi nebe 3, bet 4 reikšmes, o ta ketvirtoji yra nuo 0 iki 1, 0,001 žingsniu.

Hex kodas

Hex yra trumpinys žodžiui hexdecimal, kuris reiškia šešioliktainis. Kitaip pasakius hex kodas yra šešioliktainė skaičiavimo sistema.
Hex kodas susideda iš #RRGGBB, kiekviena raidė yra skaičius nuo 0 iki 15 aprašytas šešioliktaine sistemas ir gaunasi 0 iki f.
Pvz.:dešimtainėje 12 yra = šešioliktainėje c.
Hex kode permatamumas aprašomas kaip du papildomi simboliai gaunasi #RRGGBB.
Hex yra vienintelis spalvos apra6ymo būdas kuris gali būti trumpinamas. Pvz #RRGGBB = #rgb #RRG2PP = negali būti #RG2P

Naršyklės numatytieji stiliai

Beveik kiekvienas tagas turi savo numatytuosius naršyklės stilius.
Skirtingos naršyklės gali turėti minimalistinius skirtingus nustatymus naršyklės stiliui.
Tam, kad suvienodinti stilius tarp skirtibų naršyklių, arab netgi juos panaikinti yra naudojami normalize arba reset SCC'ai.

Normalize CSS - suvienodina stilius tarp naršyklių. Reset CSS - panaikina tagu indentitetą.

Paveldėjimas

Stiliai gali būti paveldimi iš tėvinio elemento į vaikinį elementą.

  2. Linijinis/inline
  3. Specifikacija

    3. tag 001
  4. Eiliškumas/Order

Elementui gaunant stilius iš selektorių su vienoda specifikacija yra atsižvelgiama į tų selektorių eiliškumą. Žemiau esantis selektorius su vienoda specifikacija turi aukštesnį prioritetą.

Jei įmanoma nenaudoti !Important prierašo. !Important ignoruoja visus prioritetus kitus ir suteikia nurodytą stiliu. Norint perrašyti !Important reikia naudoti kitą !Important.
