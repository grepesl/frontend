Breakpoints

Brekpoint - ekrano lūžis - kai iš vieno išdėstymo, pvz telefoninio dydžio yra pakeičiamas elementų išdėstymas į kitą išdėstymą, pvz plančetinio dydžio.
Breakpointų galima prikurti kiek tik nori (dažniausiai 3), kokiuose tiktais nori taškuose.

  1. x-small - portait phones - 0px => 575px
  2. small - landscape phone - 576px => 767px
  3. medium - tablets - 768px => 991px
  4. large - laptop - 992px => 1199px
  5. x-large - sekstop - 1200px => 1400px
  6. xx-large - large dekstop - 1400px => infinity

Tam, kad būtų sukutas vadinamasis breakpointas css reikia aprašyti jo media query.
Media wuery apsirašo tokiu būdu: @media (min-width: number_px){... CS ...}

Meida query nesuteikia jokių papildomų stiliaus prioritetų, dėl to madie query reikia rašyti teisingu eiliškumu.
Media query stiliai nepradingsta, pasiekus naują media query.

Mobile first dizaino principas yra tas, kad pradžioje puslapis yra pritaikomas mobiliesiems, o tik po to judama prie kitų ekrano dydžių.
Taip yra daroma dėl įvairių priežasčių: daugiau vartotojų naudoja mobiliuosius naršant internete; mažiau vietos informacijai - reikia atvaizduoti svarbiausią; lengviau yra pridėti turinį negu atimti...

Rašant mobile first media query, būtina naudoti min-width, ne max-width nustatymą.
