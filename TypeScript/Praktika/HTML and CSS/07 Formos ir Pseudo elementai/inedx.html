Form

Formos reikalingos norint passimti vertotojo 5vestis. Pla2iau apie tai JS temos metu.

Formos Tagai

  • form - formos tagai
  • label - formos 5vesties etiketės/pavadinimas
  • input - formos įvestis laukas (įvairus)
  • textarea - formos įvesties laukas (dideliam teksto laukui)
  • select - opcijų pasirinkimo laukas (nedideliam kiekui opcijų)
  • detalist - opcijų pasirinkimo laukas (dideliam kiekui opcijų)
  • option - opcija
  • output - formos atsakymui išvesti
  • button - mygtukas (asmeniškai nesiūlo naudoti formose)

Formos tagu atributai

  • type - input - nurodo tipą
    • text - tekstas
    • password - slapyvardis (atvaizduojamas kaip taškeliai)
    • number - skaičius
    • email - elektroninis paštas
    • url - nuoroda
    • checkbow - daugelio pasirinkims
    • radio - vieno pasirinkimas
    • submit - formos pateikimas
    • file - failo iš kompiuterio pasirinkimas
    • tel - telefonas
    • range - skaičiaus pasirinkimas slenkančioje juostoje
    • date - datos pasirinkimas (matų, mėnesio, dienos)
    • datetime-local - datos ir laiko pasirinkimas (metų, mėnesio, dienos, valandos, minutės)
    • month - mėnesio pasirinkimas
    • week - savaitės pasirinkimas
    • time - laiko pasirinkimas
    • color - spalvos pasirinkimas
    • button - mygtukas be funkcionalumo
    • image - paveiksliukas skirtas pateikti formą
    • hidden - paslėpta įvestis
    • reset - išvalo visos formos įvestis
    • search - paieškos laukas
  • for - label - nurodo kuriai 5ves2iai priskirti labelį
  • id - įvestims (input, textarea...) - apjungia labelį su įvestimi
  • name - įvestims - suteikia įvesčiai vardą (susigalvotą ir jis dažnu atveju toks pats kaip ir id)
  • value - įvestims - įvesties reikšmė
  • required - įvestim - privaloma įvesti
  • placeholder - input+textarea - pavyzdinis tekstas (laukelyje vėliau dingsta kai pradedi rašyt)
  • min - input(number/range/date)
  • max - input(number/range/date)
  • step - input(number/rage) - kitimo žingsnis
  • accept - input(file) - kokio tipo failai leistini
  • pattern - input(tel/...) - nurodo kokia turi būti įvesties seka (naudojant regex)
  • rows - textarea - eilučių kiekį kiek galima vesti (vizuolus dydis išgaunamas)
  • cols - textarea - stulpelių kiekį kiek galima vesti (vizuolus dydis išgaunamas)
  • checked - input(checkbox/radio) - iš anksto pažymėtas variantas
  • selected - option - iš anksto parinkta opcija
  • disabled - įvestis - išjungtas įvesties laukas
  • readonly - įvestims - neleidžiama keisti reikšmės
  • multiple - select - leidžia pasirinkt daug variantų
  • ...
  • ...

Pseudo Class (Form)

  • :checked - pažymėtas(checkbox/radio)
  • :default - pradinis (kur pradžioje buvo: checked arba selected)
  • :disabled - turi disabled atributą
  • :enabled - neturi disabled atributo
  • :focused - šiuo metu fokusuojama įvestis
  • :in-range - min-max ribose
  • :out-of-range - už min-max ribų
  • :indeterminate - neapibrėžta reikšmė (nesvarbu)
  • :valid - tinkama reikšmė
  • :invalid - netinkama reikšmė
  • :required - privaloma įvestis
  • :optional - neprivaloma įvestis (neturintis required atributo)
  • :readonly - elementas turi readonly atributą
  • :read-write - elementas neturi readonly atributo
