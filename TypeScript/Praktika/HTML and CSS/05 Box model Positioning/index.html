Box Model

Kiekvienas HTML elementas turi dėžutės (box), modelį, kuriame yra nurodomas informacija apie turionio, pamušalo, rėmelio, paraščių ir pozicijų dydžius.

  1. Content - turinys, centrinė dalis.
  2. Padding - pamušalas, nurodo tarpą tarp turionio ir rėmelio.
  3. Border - rėmelis, yra tarp padding ir margin
  4. Margin - paraštės, nurodo tarpą tarp rėmelio ir kitų aplinkinių elementų.
Box Modelio Pavizdys

Content

Turinys, kuris yra tekstas / media, kurį galima įvairiai stilizuoti.

Padding

Padding - tarpas tarp turinio ir rėmelio.
Jį galima nustatyti 8 būdais.

  • padding-top:
  • padding-right:
  • padding-bottom:
  • padding-left:
  • padding: all
  • padding: top/bottom right/left
  • padding: top right/left bottom
  • padding: top right bottom left

Border

Elemento rėmelis tarp margino ir paddingo.
Borderiui galima suteikti storio, stiliaus, spalvos ir kampų apvalinimo stilius. Skirtingus stilius galima suteikti skirtingoms kraštinėms ar kampams.

  • border: width type color;
  • border-width: px;
  • border-style: style;
  • border-color: color;
  • border-kraštinė: width style color
  • border-top/right/bottom/left-color: color
  • border-radius: px

Margin

Margin neveikia linijiniams elementams.

Margin - tarpas tarp rėmelio ir aplinkinių elementų.
Jį galima nustatyti 8 būdais.

  • margin-top:
  • margin-right:
  • margin-bottom:
  • margin-left:
  • margin: all
  • margin: top/bottom right/left
  • margin: top right/left bottom
  • margin: top right bottom left

Box Sizing

Nurodo kaip elemento nustatytasis plotis yra paskirstomas.
Numatytasisi nustatymas nurodo, kad prie nustatyto plo2io yra pridedami padding ir border nustatymai šitaip gaunama, kad elementas išsiplėčia.
o dažnu atveju naudojamas nustatymas (border-box) nurodo, kad nustatytą plotą paskirsto contentui paddingui ir borderiui, šitaip elementas lieka nustatyto pločio ir neišsiplėčia.

Positions

Suteikus elementui betkurią poziciją (apart numatytosios static), tuomet elemenui galima suteikti 2-index savybę, kuri suteikia galimybę kontroliuoti elemento eilės sluoksnį ekrane (kuris priekyje kuris nugaroje) ir top, right, bottom , left kurie atlieka skirtingus veiksmus priklausomai nuo pačios pozicijos.

  • Static (default)
  • Relative
  • Absolute
  • Fixed
  • Sticky
  • (ne pozicijos) z-index, top, right, bottom, left

Static

Static nustatymas nesuteikia jokių savybių, net negalima naudoti top, right, bottom, left stilių.
