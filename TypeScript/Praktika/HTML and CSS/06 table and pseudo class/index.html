Table

  • Table
  • thead - tablehead - lentelės stulpelių pavadinimai.
  • tbody - table body - lentelės duomenys.
  • tfoot - table foot - lentelės duomenų išvestinės (sumos, vidurkiai, ...)
  • tr - table row - lentelės eilutė.
  • th - table heading - lentelės stulpelio duomenų pavadinimas.
  • td - table data - lentelės duomuo.
  • caption - lentelės pavadinimas.
  • colgroup - collumn group - stulpelių grupė (bet geriau grūpuoti per ccs).
  • col - collumn - stulpelių paskirstymas.

Selectors

  • [Attribute] - atributo vardas
  • [Attribute="value"] - atributo vardas ir tiksli reikšmė
  • [Attribute~="value"] - atributo vardas ir reikšmės žodis
  • [Attribute*="value"] - atributo vardas ir tokia pati reikšmės simbolių seka
  • [Attribute^="value"] - atributo vardas ir reikšmė prasidedanti nurodyta seka
  • [Attribute$="value"] - atributo vardas ir reikšmė pasibaigoanti nurodyta seka
  • [Attribute|="value"] - atributo vardas ir reikšmė prasidedanti nurodyta reikšmė arba reikšmė-

Pseudo Class

    • :link - nelankyta nuorada
    • :visited - lankyta nuorada
    • :hover - kursorius užėjęs ant elemento (ne tik nuorodom)
    • :active - nuorados paspaudimo momentas
  • :first-child - pirmasis vaikinis
  • :last-child - paskutinis vaikinis
  • :only-child - vienintelis vaikinis
  • :first-of-type - pirmasisi tokio tipo vaikinis
  • :last-of-type - paskutinis tokio tipo vaikinis
  • :only-of-type - vienintelis tokio tipo vaikinis
  • :nth-child(n) - n'tąjį vaikinį
  • :nth-last-child(n) - n'tąjį vaikinė nuo galo
  • :nth-of-type(n) - n'tąjį tokio pat tipo
  • :nth-last-of-type(n) - n'tąjį tokio tipo vaikinį nuo galo
  • (skaičius) kelintą, (skaičius n) kas kelintą, (skaičius n + skaičius) nuo kurios kas kelintą
    • :root
    • :not(selector)
    • :is()
    • :has()
    • :where()
    • ...
