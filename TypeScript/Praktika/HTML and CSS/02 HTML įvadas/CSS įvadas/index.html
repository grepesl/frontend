CSS Įvadas

  • Inline - linijinis - kai CSS kodas rašomas tago atribute
  • Internal - vidinis - kai CSS kodas rašomas HTML faile, head tago style tago viduje
  • external - išorinis - kai CSS kodas rašomas atskirame CSS faile, kuris yra prijungiamas prie HTML naudojant link tagą

Inline

Inline CSS rašymo būdas yra naudojamas norint pasitestuoti kažkokius stilius.
Norint panaudoti inline css rašymo būdą, reikia tagui suteikti style atributą, kurio viduje rašome stilius.

External

Išorinis CSS rašymo būdas yra naudojamas dažniausiai.

Selektoriai

Selektoriai naudojami tam, kad CSS ra6omi stiliai galėtų būti priskirti tam tikriems HTML elementams.

  • tag selektorius - nurodomas tago
  • multiple - per kablelį galima pasirinkt kelis selektorius
  • * - visko selektorius
  • .class - susigalvojame klases pav patys
  • #id - susigalvojami id pav patys

Class

Aprašant klasę CSS pradedame nuo to kad parašome . kuris indikuoja CSS kad bus kuriama klasė tuomet po taško rašome sugalvotą pav be tarpo ir liet raidžių.

ID

Apra6ant id CSS pradedame nuo to, kad para6ome groteles kurios indikuoja CCS kad bus kuriama ID. tuomet po groteliu ra6ome susigalvota pavadinima (be tarpu ir lietuvisku raidziu)
Norint priskirti id HTML reikia elementui suteikti id atribute ir nurofyti norimo id pav.
