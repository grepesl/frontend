Transform

tRANSFORM YRA stilius, kuris nusako kad elementas bus kažkaip pakeičiamas, o visi žemiau išvardinti dalykai yra šio stiliaus nustatymai.

  • rotate(Z) - sukasi aplink Z ašį
  • rotateZ(Z) - sukasi aplink Z ašį
  • rotateX(X) - sukasi aplink X ašį
  • rotateY(Y) - sukasi aplink Y ašį
  • translate(X) - pastūmimas X ašyje
  • translate(Y) - pastūmimas Y ašyje
  • translate(X, Y) - pastūmimas X ir Y ašyse
  • scale(X) - keičia elemento dydį X ašyje
  • scale(Y) - keičia elemento dydį Y ašyje
  • scale(X, Y) - keičia elemento dydį X ir Y ašyse. Parašius vieną rodmenį/argumentą jis taikomas abejoms ašims

Rotate, translate ir scale nustatymai jau turi savo atskyrus stilius ir tamkad juos panaudotume nebebutina rasyti kaip funkciju trandform stiliaus viduje.

Transform-origin

Transform origin nusako nuo kurios elemento vietos vyksta rotate, scale ir skew kitimai.
Transform-origin default yra elemento centre.

Transition

Kontroliuoja stilių perėjimą iš vieno į kitą.
Tranzicija dažniausiai yra daroma kai ant elemento yra užeinamas pelės kursorius arba kaqi jis yra paspaudžiamas.
Tranzicijos kitimas vyksta vieną sykį iš vieno stiliaus į kitą stiliu ir viskas.

  • transition-duration: seconds
  • transition-timing-function: timin-function
  • transition-delay: seconds (defailt - 0s)
  • transition-property: style
  • transition-behavior: normal(default) | allow-discrete;
  • transition: transition-duration transition-delay transition-timing-function transition-property transition-behavior

Animation

kontroliuoja stilių perėjimą iš vieno į kitą
Animacijai nebūtina vartotojo unterakcija ir ji gali vykti kiek tik nori artų.
Animacijos metu tie patys stiliai gali turėti daugelį skirtingų nustatymų skirtingu animacijos metu.

  • animation-duration: seconds
  • animation-delay: seconds
  • animation-timing-function: timing-function
  • animation-name: susigalvotas vardas
  • animation-iteration-count: number | infinite
  • animation-direction: normal | reverse | alternate(nuo pradzios i pabaiga ir tada nuo pabaigos i pradzia) | reverse-alternate
  • animation-play-state: running | paused
  • animation-fill-mode: none | forwards | backwards | both
  • animation-composition: replace | add | accumulate
  • animation: combination (visų apjungimas) (duration -> delay, o kiti nesvarbu eiliškumas)

Tam, kad animacijos nustatymai būtų reikšmingi, reikia pačios animacijos.
O pati animacija aprašoma kryframes pagalba.
@keyframe susigalvotas vardas { 0%{} n%{} n+1%{} ... 100%{} }
